# Set-API

## Requirements

1. ruby (2.2.0)
2. bundler (1.0 or greater)
3. postgres (9.1.0 or greater)

## Installation

1. Install ruby 2.2.0 with your version manager of choice.
2. Install the project's gems with bundler.

## Running the app

Assuming Postgresql is already running:

`rails server`
